# خطة تنفيذ تصميم الواجهة الرئيسية (الهيدر والبطاقات)

بما أننا حصلنا على خط **(Diwani Bent)** الرائع، وسنستخدم حزمة الأيقونات **`flutter_islamic_icons`**، فقد أصبح بإمكاننا الاستغناء عن الصور تماماً وبناء البطاقات برمجياً بنسبة 100%. 
هذا سيعطيك واجهة **خفيفة جداً، وسريعة، وتتمدد بشكل مثالي** على جميع الشاشات.

إليك الخطة التنفيذية المعمارية لبناء هذه الشاشة:

---

## 🛠️ المرحلة الأولى: إعداد الموارد (Assets Setup)

قبل كتابة أي كود، يجب تجهيز المشروع ليتعرف على الخط والصور:
1. قم بإنشاء مجلد للخطوط `assets/fonts/` وانقل ملف الخط (مثلاً `DiwaniBent.ttf`) من مجلد `arfonts-diwani-bent` إليه.
2. تأكد من وجود ملف `header_pattern.jpg` (الذي استخرجناه سابقاً) في مجلد `assets/images/`.
3. أضف التعريفات في ملف `pubspec.yaml`:

```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_islamic_icons: ^1.0.3  # حزمة الأيقونات

flutter:
  assets:
    - assets/images/header_pattern.jpg
  fonts:
    - family: DiwaniBent
      fonts:
        - asset: assets/fonts/DiwaniBent.ttf
```

---

## 🎨 المرحلة الثانية: برمجة الواجهة الرئيسية (Main Screen Layout)

سنستخدم بنية `CustomScrollView` مع `Slivers` لأنها الأفضل احترافياً للتعامل مع هيدر (AppBar) كبير وشبكة (Grid) أسفله.

### 1. تصميم الهيدر (The Header)
سنستخدم `SliverAppBar` لكي يظهر الهيدر بشكل فخم ويختفي تدريجياً عند النزول للأسفل (Scrolling).

```dart
SliverAppBar(
  expandedHeight: 120.0, // ارتفاع الهيدر
  floating: false,
  pinned: true,
  backgroundColor: const Color(0xFF3F1D20), // لون عنابي داكن كخلفية
  flexibleSpace: FlexibleSpaceBar(
    background: Container(
      decoration: const BoxDecoration(
        image: DecorationImage(
          image: AssetImage('assets/images/header_pattern.jpg'),
          fit: BoxFit.cover, // لتغطية الشاشة أفقياً بدون تشوه
        ),
      ),
    ),
  ),
)
```

---

### 2. تصميم البطاقة المبرمجة (The Custom Card)
كل بطاقة عبارة عن `Container` يحتوي على تدرج زجاجي (Gradient) وظل، وبداخله `Column` يضم الأيقونة والنص بخط (Diwani Bent).

```dart
Widget _buildQuranCard({
  required String title,
  required IconData icon,
  required VoidCallback onTap,
}) {
  return InkWell(
    onTap: onTap,
    borderRadius: BorderRadius.circular(12),
    child: Container(
      decoration: BoxDecoration(
        // التدرج اللوني الذي يعطي لمعة زجاجية خفيفة
        gradient: const LinearGradient(
          colors: [Color(0xFFFFFFFF), Color(0xFFE8E8E8)],
          begin: Alignment.topCenter,
          end: Alignment.bottomCenter,
        ),
        borderRadius: BorderRadius.circular(12),
        // الظل السفلي لتبدو البطاقة بارزة
        boxShadow: [
          BoxShadow(
            color: Colors.black.withOpacity(0.1),
            blurRadius: 8,
            offset: const Offset(0, 4),
          ),
        ],
      ),
      child: Column(
        mainAxisAlignment: MainAxisAlignment.center,
        children: [
          // 1. الأيقونة الإسلامية
          Icon(
            icon,
            size: 45,
            color: const Color(0xFF5A2A2E), // اللون العنابي/البني الداكن
          ),
          const SizedBox(height: 12),
          // 2. النص بالخط الديواني
          Text(
            title,
            textAlign: TextAlign.center,
            style: const TextStyle(
              fontFamily: 'DiwaniBent', // استخدام الخط الذي أضفناه
              fontSize: 22,
              color: Color(0xFF5A2A2E),
            ),
          ),
        ],
      ),
    ),
  );
}
```

### 3. تجميع الواجهة (Grid Layout)
الآن نضع البطاقات داخل `SliverGrid` لنضمن أنها تترتب في عمودين ولا تتشوه في الشاشات العريضة.

```dart
SliverPadding(
  padding: const EdgeInsets.all(16.0),
  sliver: SliverGrid(
    gridDelegate: const SliverGridDelegateWithFixedCrossAxisCount(
      crossAxisCount: 2, // عمودين
      crossAxisSpacing: 16.0, // المسافة الأفقية
      mainAxisSpacing: 16.0, // المسافة العمودية
      childAspectRatio: 1.4, // نسبة العرض للطول (مهم جداً للحفاظ على شكل البطاقة)
    ),
    delegate: SliverChildListDelegate([
      _buildQuranCard(
        title: 'قراءة القرآن',
        icon: FlutterIslamicIcons.quran2, // أيقونة المصحف
        onTap: () {},
      ),
      _buildQuranCard(
        title: 'استماع للقرآن',
        icon: FlutterIslamicIcons.solidQuran2, // أيقونة استماع/مصحف
        onTap: () {},
      ),
      _buildQuranCard(
        title: 'أذكار الصباح',
        icon: FlutterIslamicIcons.crescentMoon, // أيقونة هلال
        onTap: () {},
      ),
      // ... باقي البطاقات ...
    ]),
  ),
)
```

## 🎯 النتيجة النهائية للبرمجة (لماذا هذا الحل عبقري؟)
1. **وزن خفيف (0 KB صور للبطاقات):** الواجهة كاملة مرسومة بالكود، حجم التطبيق سيكون صغيراً جداً.
2. **استجابة مثالية (Responsive):** لن تنضغط البطاقات في الشاشات الصغيرة بفضل `childAspectRatio`.
3. **فخامة ديوانية:** استخدام خط `DiwaniBent` الحقيقي يعني أن النص سيظل حاداً جداً (Crisp) حتى لو كبرت الشاشة لـ iPad أو شاشة 4K.
4. **ألوان متناسقة:** الكود يستخدم الـ Hex Colors المأخوذة من الصورة المرجعية (درجات العنابي الداكن `0xFF5A2A2E` والأبيض الزجاجي).
